Методы модуля​

НовоеСообщение​

Конструктор сообщения, получаемого от реактивного приложения

Возвращает

Структура -

  • ИмяСобытия - Строка - для ветвления по событиям
  • Ид - Строка - Идентификатор сообщения
  • Значение - Структура - параметры событияМассив - параметры событияЧисло - параметры событияСтрока - параметры событияБулево - параметры событияДвоичныеДанные - параметры событияНеопределено - параметры события

ОбработатьСообщение​

Перехватыет сообщение из события *ПриНажатии поля HTTP документа и формирует обработку оповещения

Параметры метода

  • ЭтотОбъект
    • ФормаКлиентскогоПриложения - Форма с которой нужно взаимодействовать
  • ЭлементПолеHTML
    • ПолеФормы - ПолеHTMLДокумента в которое загружено реактивное приложение
  • ДанныеСобытия
    • ФиксированнаяСтруктура - См. описание ПриНажатии в синтакс-помощнике
  • СтандартнаяОбработка
    • Булево - Возвращаемый. Устанавливливается в Ложь если событие реактивного приложения.
  • Обработчик
    • ОписаниеОповещения - В результат будет возвращен объект См. НовоеСообщение

ОтправитьОтвет​

Ответ на сообщение от реактивного приложения. Каждое сообщение должно получить свой ответ! Хотябы пустой.

Параметры метода

  • ЭлементПолеHTML
    • ПолеФормы - ПолеHTMLДокумента в которое загружено реактивное приложение
  • Сообщение
    • НовыйСообщениеСм. НовыйСообщение - Сообщение, на которпое готовится ответ
  • Ответ
    • Структура - Ответ, который будет отправлен
    • Массив - Ответ, который будет отправлен
    • Число - Ответ, который будет отправлен
    • Строка - Ответ, который будет отправлен
    • Булево - Ответ, который будет отправлен
    • Неопределено - Ответ, который будет отправлен
